### Dynamic Font size

#### <h3>In Percentage</h3>

<P>In a web browser 100% font size = 16px</P>

- e.g If you need a font size of 90px then 90/16 * 100 = 562.5%

- <P>em (phonetic pronunciation of letter M)</P>

 - <P>em is the width of the capital letter M</P>

- In a web browser 1em font size = 16px</P>

<P>e.g If you need a font size of 90px then 90/16 = 5.625em</P>

<h4>rem (root em)</h4>

- <P>rem ignores all of the parent settings for the font size and
set the new font size relative to the root ( or the default font size).</P>
